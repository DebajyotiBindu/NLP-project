## Role Predictor App
A machine learning app that predicts the most suitable job role (like Data Analyst, ML Engineer, Web Developer, etc.) based on a resume’s text — such as listed skills, tools, and experience.
Built using TF-IDF for text representation and Multinomial Naive Bayes for classification.

### Features:
* Predicts job role based on resume text or skills.
* Accepts variable-length resume text (no word limit).
* Built with interpretable NLP model (see top keywords per role).
* Deployed using Streamlit for a clean and interactive UI.
